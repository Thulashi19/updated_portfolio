Green PlantShop

Green is a responsive online plant shop built using the MERN stack (MongoDB, Express, React, Node.js). It allows users to explore and order a variety of indoor and outdoor plants easily.

Features

  • 1) 🌿 User-Friendly Interface – Clean, modern design with responsive layout.
  • 2) 🛒 Plant Catalog – View different plant types with names, prices, and images.
  • 3) 🔍 Product Details – Learn more about each plant before buying.
  • 4) 🧾 Add to Cart – Seamless cart management with total price tracking.
  • 5) 💳 Payment Page – Simple and intuitive payment confirmation.
  • 6) 🔐 Login/Register – Secure user authentication built with Node and MongoDB.
